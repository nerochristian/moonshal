Place payment panel assets in this folder.

Supported banner filenames:
- paypanel_banner.png
- paypanel_banner.jpg
- paypanel_banner.jpeg
- server_banner.png
- server_banner.jpg
- server_banner.jpeg
- banner.png
- banner.jpg
- banner.jpeg

Supported QRIS filenames:
- qris.png
- qris.jpg
- qris.jpeg
- payment_qris.png
- payment_qris.jpg
- payment_qris.jpeg

Set PAYPAL_URL in .env to enable the PayPal link button response.
